# Student Management System

## 📌 Project Description

The Student Management System is a full-stack web application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). It provides an easy way to manage students, courses, attendance, and examination results through a modern web interface.

---

## 🚀 Features

* User Login & Registration
* Dashboard with Statistics
* Student Management (CRUD)
* Course Management (CRUD)
* Attendance Management (CRUD)
* Result Management (CRUD)
* Search Functionality
* PDF Export
* Excel Export
* Responsive User Interface
* Dashboard Charts

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Renuka-9059/StudentManagementSystem.git
```

### Install dependencies

#### Client

```bash
cd client
npm install
```

#### Server

```bash
cd ../server
npm install
```

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
cd ../client
npm start
```
